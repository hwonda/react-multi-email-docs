# Essential Props of react-multi-email

The react-multi-email component is versatile and customizable.

However, there are **three core props** that you must understand to utilize it effectively<br/>
: `emails`, `onChange`, and `getLabel`.

## emails

The `emails` prop represents **the array of email addresses**.

- Property: `emails`
- Type: `string[]`
- Default: `[]`

:::info
This prop is essential for tracking the state of the email addresses entered by the user.
:::

## onChange
The `onChange` prop is a callback function that **receives the updated email array whenever a change occurs**.

- Property: `onChange`
- Type: `(emails: string[]) => void`
- Default: `() => {}`

:::info
Use this callback to update the state of your emails, ensuring your component stays up-to-date with user input.
:::

## getLabel

The `getLabel` prop is a function that **allows customization of how each email tag is displayed**.

- Property: `getLabel`
- Type: `(email: string, index: number, removeEmail: (index: number) => void) => ReactNode`
- Default: `(email, index, removeEmail) => email`

:::info
This prop gives you the flexibility to **style** each email tag, adding a personal touch to your component's appearance.
:::
