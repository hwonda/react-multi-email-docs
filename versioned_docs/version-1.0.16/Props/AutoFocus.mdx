## Value

- Property: ```autoFocus```<br/>
- default: ```{false}```<br/>
- type: ```boolean``` 

## Result

<AutoFocus />

## Code

```tsx title="ReactMultiEmail.tsx"
<ReactMultiEmail 
    autoFocus={true}
/>
```

AutoFocus is a feature that focuses on the text area when the page is first loaded. 

If it is set to true as shown in the above code, it will be automatically focused.
<br/><br/>

:::warning Must check!
The current page is configured with mdx and may not appear to be working. To check the function, set it up directly in demo or reload the page via 'F5' or 'Ctrl(Command)+Shift+R'.
:::
